# Cursor Navigation

Navigate UI elements using your controller or keyboard without a mouse.

## How It Works

The cursor navigation system allows you to:
- Select UI elements (buttons, items, NPCs)
- Navigate between interactive frames
- Interact with game world objects

## Controls

| Action | Default Binding |
|--------|-----------------|
| Move Cursor | D-Pad / Arrow Keys |
| Select/Click | A Button / Enter |
| Back/Cancel | B Button / Escape |
| Next Frame | Tab |
| Previous Frame | Shift+Tab |

## Supported Frames

The cursor can navigate:
- Bags and inventory
- Character panel
- Spell book
- Quest log
- Vendor windows
- Bank
- Mail
- Trade windows
- Most standard UI elements

## Tooltip Display

When hovering over items or elements, the tooltip appears showing:
- Item stats and info
- Spell descriptions
- NPC information
- Action button contents

## Tips

- The cursor highlights the currently selected element
- Use the crosshair to target enemies/NPCs in the world
- Some custom addon frames may not be fully supported
- Press Escape to clear cursor selection
