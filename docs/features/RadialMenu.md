# Radial Menu

Quick access wheel menu for common actions and menus.

## Opening the Menu

The radial menu can be opened with a configured keybind (check Keybindings settings).

## Menu Options

The radial menu provides quick access to:
- Character Panel
- Spell Book
- Talents
- Quest Log
- Social/Friends
- Map
- Bags
- Main Menu

## Navigation

1. Open the radial menu with your keybind
2. Use the D-Pad or stick to highlight an option
3. Press A/Select to choose
4. Press B/Cancel to close without selecting

## Customization

The radial menu layout is pre-configured for common actions. Menu items correspond to the standard WoW interface panels.

## Tips

- Great for quickly opening panels without keyboard shortcuts
- Works well with controller navigation
- Faster than navigating through the main menu
