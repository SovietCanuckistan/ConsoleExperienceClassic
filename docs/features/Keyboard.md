# Virtual Keyboard

On-screen keyboard for typing in chat and other text fields without a physical keyboard.

## Activation

The virtual keyboard automatically appears when:
- You click the chat frame to type
- You open a text input field
- Chat edit box becomes visible

## Layout

The keyboard displays a full QWERTY layout with:
- Letter keys (A-Z)
- Number keys (0-9)
- Common punctuation
- Space bar
- Enter/Send
- Backspace
- Shift for capitals

## Navigation

| Action | Control |
|--------|---------|
| Move between keys | D-Pad / Arrow Keys |
| Press selected key | A Button / Enter |
| Backspace | B Button |
| Send message | Start / Enter on keyboard |
| Close keyboard | Escape |

## Chat Integration

When the keyboard is open:
1. The chat frame expands to show more messages
2. Type your message using the on-screen keyboard
3. Press Enter/Send to send the message
4. Press Escape to close without sending

## Configuration

| Setting | Description | Location |
|---------|-------------|----------|
| Enable Keyboard | Show virtual keyboard for chat | Chat Settings |

## Tips

- The keyboard supports both uppercase and lowercase
- Use Shift key for capital letters
- Great for quick chat messages with a controller
- For long typing sessions, consider using a physical keyboard
